# 面试文档导入格式规范

## 文件命名格式

```
{公司名} - {部门/方向} - {面试轮次}.md
```

**示例：**
- `美团 - 本地核心商业 - 一面.md`
- `美团 - 本地核心商业 - 二三面模拟.md`
- `腾讯PCG - 智能营销AI小游戏 - 面试题.md`

## 文档内容格式

### 题目格式
```markdown
## 1. 问题内容
```

### 答案格式
```markdown
## 答案

答案内容...

---

```

## 格式要求

| 元素 | 格式规范 |
|------|----------|
| 标题 | `# 公司名 - 部门 - 轮次` |
| 题目 | `## 序号. 问题` |
| 答案 | `## 答案` 开头，后跟内容 |
| 分隔 | 题目间用 `---` 分隔 |

## 答案内容支持格式

- **加粗**：`**文字**`
- **列表**：`- ` 或 `1. `
- **表格**：使用 Markdown 表格语法
- **代码块**：```语言 代码```
- **引用**：`> 引用内容`

## 导入目标

面试文档应存放于 `src/data/interview/` 目录，按公司名称创建对应的数据文件。

例如：
- 美团 → `src/data/interview/meituan-local-commerce.ts`
- 腾讯 → `src/data/interview/tencent-pcg.ts`
