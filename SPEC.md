# 前端面试备考网站 - 开发提案

## 1. 项目概述

### 项目名称
**InterviewFlash** - 前端面试备考记忆卡片系统

### 核心价值
将面试核心考点、项目复盘、算法刷题整合为交互式记忆卡片，辅助候选人高效备考。

### 目标用户
- 谭成（候选人）- 研二学生，滴滴实习经验
- 具备 React + TypeScript + Vite 技术栈背景

---

## 2. 产品功能规划

### 2.1 三大核心模块

| 模块 | 内容 | 卡片数量 |
|------|------|----------|
| **前端基础核心考点** | JavaScript/TypeScript/React/浏览器/工程化等10个章节 | ~200张 |
| **项目针对性复盘** | 滴滴实习（重构/AI工程化/性能优化）+ GResume（CRDT/离线/AI集成）| ~50张 |
| **刷题模块** | 手撕代码/场景设计/概念解释 按类型分类 | ~100道 |

### 2.2 交互功能

| 功能 | 描述 |
|------|------|
| 卡片翻转 | 点击卡片显示答案/解析 |
| 进度追踪 | 显示已掌握/待复习/未开始 |
| 随机练习 | 随机抽取卡片进行练习 |
| 分类浏览 | 按章节/难度/状态筛选 |
| 搜索 | 关键词搜索卡片 |
| 收藏 | 标记重点卡片便于复习 |

---

## 3. 技术选型

### 核心技术栈

| 技术 | 选择理由 |
|------|----------|
| **React 18** | 候选人熟悉，生态完善 |
| **TypeScript** | 类型安全，面试备考本身需要严谨 |
| **Vite** | 候选人熟悉，构建速度快 |
| **Tailwind CSS** | 快速开发，响应式友好 |
| ** Zustand** | 状态管理，候选人简历中提到过 |

### 第三方库

| 库 | 用途 |
|----|------|
| `framer-motion` | 卡片翻转动画 |
| `@dnd-kit` | 拖拽排序（如需要）|
| `react-markdown` | 渲染解析文本 |
| `lucide-react` | 图标库 |
| `clsx` / `tailwind-merge` | 动态class |

## 6. 数据结构设计

### 6.1 卡片基础结构

```typescript
interface FlashCard {
  id: string;
  module: 'core' | 'projects' | 'algorithms';
  chapterId: string;
  category?: string;

  // 内容
  question: string;           // 问题
  answer: string;            // 答案
  tags: string[];            // 标签

  // 掌握程度
  status: 'unvisited' | 'forgotten' | 'fuzzy' | 'mastered';

  // 扩展
  codeExample?: string;       // 代码示例
  extendQuestion?: string;   // 延伸追问
  difficulty?: 'easy' | 'medium' | 'hard';
}
```

### 6.2 章节结构

```typescript
interface Chapter {
  id: string;
  module: 'core' | 'projects' | 'algorithms';
  title: string;
  description?: string;
  cardCount: number;
  cards: FlashCard[];
}
```

### 6.3 学习进度

```typescript
interface LearningProgress {
  userId: string;
  cardProgress: Record<string, CardProgress>;  // cardId -> progress
  lastVisited: {
    cardId: string;
    timestamp: number;
  };
  totalMastered: number;
  totalCards: number;
}

interface CardProgress {
  status: 'unvisited' | 'forgotten' | 'fuzzy' | 'mastered';
  lastReviewed?: number;
  reviewCount: number;
}
```

## 7. 技术实现细节

### 7.1 卡片翻转动画

使用 `framer-motion` 实现 3D 翻转效果：

```tsx
<motion.div
  initial={false}
  animate={{ rotateY: isFlipped ? 180 : 0 }}
  transition={{ duration: 0.6 }}
  style={{ transformStyle: 'preserve-3d' }}
>
  <div className="front">
    <Question />
  </div>
  <div className="back" style={{ transform: 'rotateY(180deg)' }}>
    <Answer />
  </div>
</motion.div>
```

### 7.2 状态管理

使用 Zustand 管理全局状态：

```typescript
// store/useCardStore.ts
interface CardStore {
  cards: FlashCard[];
  currentCard: FlashCard | null;
  filter: FilterOptions;

  // Actions
  flipCard: () => void;
  setCardStatus: (id: string, status: CardStatus) => void;
  nextCard: () => void;
  prevCard: () => void;
}
```

### 7.3 本地存储

使用 `localStorage` 持久化学习进度：

```typescript
// 进度持久化
useEffect(() => {
  localStorage.setItem('progress', JSON.stringify(progress));
}, [progress]);

// 初始化读取
const savedProgress = localStorage.getItem('progress');
```
