# 实习经历

## 滴滴企业版 - 前端开发实习生

- **时间**：2024.07 - 2025.08
- **部门**：商旅体验
- **岗位**：前端开发实习生

### 项目描述

滴滴企业版商旅一站式管理平台，承载机票、酒店、火车票等全品类差旅业务，覆盖 57 个业务模块运营管理

### 核心职责与产出

**【核心架构重构】**

- 参与遗留系统的拆解与重构，抽象通用定价模型并将硬编码逻辑转化为配置驱动机制，降低业务耦合度。
- 自研基于 JSON Schema 的动态表单引擎，实现复杂业务场景的零代码快速扩展与配置化生成，大幅缩短迭代周期并显著降低多业务线维护成本。

**【企业级基建与性能优化】**

- 搭建高内聚业务组件体系，统一网络请求与错误处理机制，消除大量冗余代码。
- 针对弱网场景实施路由分割、预加载及构建优化，消除冗余依赖，显著降低 FCP 并提升核心页面交互速度。

**【AI 工程化转型与实践】**

- 参与推动团队 AI Native 转型，基于 OpenSpec 制定智能体（SSD）定义与 Skills 封装标准，规范大模型接入。
- 推广 AI 结伴编程模式，利用 AI 辅助生成标准化组件样板代码及重构建议，实现从小型需求到核心模块重构的全流程提效，推动团队研发规范自动化落地。
