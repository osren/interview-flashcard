# 项目经验

## GResume 智能简历编辑一体化平台

描述: 一款端到端的智能简历解决方案，深度融合离线优先架构、多人实时协作与 AI ATS 智能评估能力。平台涵盖简历创建、智能优化、投递追踪全流程，旨在通过前沿技术栈提供极致流畅的编辑体验与数据安全保障。

项目亮点: 实时协作系统：基于 Automerge (CRDT) 构建四层同步架构解决并发冲突，设计 Zustand（乐观更新 → CRDT 同步 → 延迟持久化）三步模型，结合 IndexedDB 实现离线编辑与断网续传，确保数据零丢失。
性能极致优化：实施乐观 UI 实现操作"低延迟"响应，通过路由级代码分割与光标节流批处理大幅降低加载时间与网络负载，结合原子化 CSS 深度优化渲染性能。
AI 智能化集成：对接 DeepSeek LLM 构建多维度 ATS 评分系统，基于 Supabase Edge Functions 实现安全流式调用，提供简历与 JD 的智能对比及可视化优化建议。

Github: https://github.com/506-FETL/resume
