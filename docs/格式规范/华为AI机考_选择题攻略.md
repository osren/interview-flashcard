## 📐 1. 线性代数

### 核心要点

**矩阵基础**
- **秩（Rank）**：矩阵中线性无关的行/列的最大数目
- **行列式 det(A)**：为 0 ↔ 矩阵奇异（不可逆）；不为 0 ↔ 可逆
- **迹（Trace）**：主对角线元素之和，等于特征值之和
- **单位矩阵 I**：主对角线全为 1，行列式 = 1，一定可逆，逆矩阵是自身

**向量空间**
- **零空间（Null Space）**：满足 Ax = 0 的所有向量 x 构成的空间（即核空间）
- **列空间**：A 的列向量张成的空间
- **行空间**：A 的行向量张成的空间
- **左零空间**：满足 A^T x = 0 的向量空间（与列空间正交）

**特征值与特征向量**
- **定义**：Av = λv，λ 为特征值，v 为特征向量
- **对称矩阵**：特征值均为实数，特征向量互相正交
- **正定矩阵**：所有特征值 > 0

**奇异值分解（SVD）**
- **公式**：A = UΣV^T
  - U（m×m）：左奇异向量矩阵（列正交）
  - Σ（m×n）：对角矩阵，对角线为奇异值（非负，降序排列）
  - V（n×n）：右奇异向量矩阵（列正交）
- **矩阵的秩** = Σ 对角线上**非零奇异值**的个数
- **应用**：图像压缩/去噪（低秩近似）、文本主题建模（LSA）、推荐系统

**矩阵运算**
- 矩阵乘法维度规则：(m×n)·(n×k) = (m×k)，内维 n 消去
- 线性映射 Y = XW，X∈ℝ^(n×d)，W∈ℝ^(d×k)，则 Y ∈ ℝ^(n×k)
- **余弦相似度**：cos θ = (a·b) / (|a|·|b|)，范围 [-1, 1]

**矩阵条件数**
- 定义：κ(A) = ‖A‖·‖A^(-1)‖
- 条件数越大，矩阵越"病态"，数值求解越不稳定

**Newton-Schulz 迭代（矩阵求逆）**
- 迭代公式：X_{k+1} = X_k(2I - AX_k)
- 误差矩阵 E_k = I - AX_k，可推出 E_{k+1} = E_k^2（二次收敛）
- 收敛条件：谱半径 ρ(E_0) < 1

**正交变换**
- 正交矩阵 Q 满足 Q^T Q = I
- 正交变换**保持不变**：内积、欧氏距离（L2范数）、余弦相似度
- 正交变换**不保持**：曼哈顿距离（L1范数）

## 📊 2. 概率论与数理统计

### 核心要点

**基本概念**
- **期望线性性**：E(aX + bY + c) = aE(X) + bE(Y) + c（无需独立）
- **方差**：Var(X) = E(X²) - [E(X)]² ≥ 0
- **Jensen 不等式**：[E(X)]² ≤ E(X²)（方差非负的推论）
- **E(XY) = E(X)·E(Y)** 仅在 X, Y 不相关时成立，相关时一般不等

**条件概率与独立性**
- P(AB) = P(A)·P(B|A)（当 P(A)>0 时成立）
- 独立性定义：P(AB) = P(A)P(B) ↔ P(A|B) = P(A)
- 注意：两两独立 ≠ 相互独立（A与BC独立需满足更强条件）

**正态分布最大似然估计（MLE）**
- 均值 μ 的 MLE：μ̂ = x̄（样本均值）—— **无偏估计**
- 方差 σ² 的 MLE：σ̂² = (1/n)∑(xi - x̄)²  —— **有偏估计**（无偏版本除以 n-1）

**相关系数**
- r = 1：完全正线性相关
- r = -1：完全负线性相关（y = ax + b，a < 0）
- r = 0：线性不相关（但可能有非线性关系）

**二项分布**
- P(X = k) = C(n,k)·p^k·(1-p)^(n-k)

**插值多项式**
- 插值条件：多项式在节点处精确等于给定值
- n+1 个互异节点 → 唯一确定次数不超过 n 的多项式
- Lagrange 插值 = Newton 插值（同一多项式不同表达形式）
- 高次插值不一定更好，会出现 **Runge 现象**（端点附近振荡）

## 🤖 3. 机器学习基础

### 核心要点

**K-Means 聚类**
- 必须预先指定聚类数 **K**
- E步：按欧氏距离分配样本到最近质心
- M步：更新质心为簇内均值
- 初始化方法：K-Means++ 改进随机初始化

**线性回归**
- 决定系数 R² = 1 - SS_res/SS_tot，范围 [0, 1]，越接近 1 拟合越好
- 增加特征（哪怕无关）R² 不会降低（因此需用调整 R²）
- 不同数据集的 R² 不可直接比较

**正则化**
- **L1 正则化（Lasso）**：产生稀疏解，可自动特征选择
- **L2 正则化（Ridge）**：防止过拟合，提高泛化能力

**评价指标**
- 召回率 Recall = TP/(TP+FN)
- 精确率 Precision = TP/(TP+FP)
- 准确率 Accuracy = (TP+TN)/(TP+TN+FP+FN)
- TPR = Recall；特异度（Specificity）= TN/(TN+FP)
- **Micro-F1**：汇总所有类别的 TP/FP/FN，样本多的类别权重更大
- **Macro-F1**：各类别 F1 简单平均，平等对待各类别

**特征工程**
- **特征降维**（如 PCA）：通过变换减少维度
- **特征选择**：挑选特征子集，不改变维度
- **互信息（MI）**：最稳健的特征相关性指标，不假设线性关系，适合非线性、非正态场景

**距离度量**
- **余弦相似度**：适合高维稀疏向量（文本、推荐系统）
- **欧氏距离**：最常用向量距离
- **曼哈顿距离**：L1范数，适合稀疏特征
- **汉明距离**：适合二进制/离散特征
- **KL散度**：度量两个概率分布的差异（非对称）

**层次聚类（HAC）**
- **Complete Linkage（全链接）**：簇间距离 = 两个簇最远点对的距离
- **Single Linkage（单链接）**：簇间距离 = 两个簇最近点对的距离

**线性判别分析（LDA）**
- **有监督**降维（≠ 无监督）
- 目标：最大化类间散度/类内散度
- 降维后维度最多为**类别数-1**
- 假设：各类高斯分布，协方差矩阵相同

**非负矩阵分解（NMF）**
- 适用场景：数据非负且需要可解释性（如文本聚类）
- 要求输入矩阵和分解因子均非负

## 🧠 4. 深度学习与神经网络

### 核心要点

**神经网络基础**
- 相比线性模型核心优势：**可以学习非线性关系**（通过激活函数）
- 输入层：不含可训练参数，只传递数据
- 输出层：神经元数由任务决定（二分类1、多分类C、回归1）
- 各层神经元数量**不需要相同**

**激活函数**
- **ReLU**：f(x) = max(0, x)
  - Dead ReLU 问题：输入 < 0 时梯度为 0，权重无法更新
  - 学习率过大可能导致神经元永久死亡
  - 死亡神经元有可能因上游权重变化而"复活"（D错）
- **Leaky ReLU**：可缓解 Dead ReLU 问题
- **Sigmoid / Softmax**：输出归一化概率

**Softmax**
- 输入相同（如 [1,1,1]）→ 均匀分布（如 [1/3, 1/3, 1/3]）

**损失函数**
- **MSE**：对异常值敏感
- **MAE**：对异常值鲁棒，但在零点不可导
- **Huber Loss**：小误差用平方损失（光滑），大误差用线性损失（鲁棒），处处可导

**归一化层**
- **Batch Normalization（BN）**：按 mini-batch 归一化
- **Layer Normalization（LN）**：按单样本所有通道归一化；分 Pre-LN 和 Post-LN
- **RMSNorm**：去掉 LN 中减均值的部分（只按 RMS 缩放）
  - 注意：是 **RMSNorm** 去掉了减均值，而非 LN

**反向传播**
- 反向传播计算量约为前向传播的 **2 倍**
- 需要存储所有中间层的激活值 → 显存消耗大
- 使用**自动微分（AD）**，非严格符号/数值微分

**优化器**
- **SGD**：全局统一学习率，非自适应
- **Adam**：维护一阶矩 m 和二阶矩 v，自适应缩放学习率
- **AdamW**："W"代表**解耦权重衰减（Weight Decay decoupling）**
  - 将权重衰减从梯度更新中解耦，正则化效果更稳定
  - 自适应学习率机制不变
- **RMSprop**：维护梯度二阶矩，自适应
- Adam/AdamW/RMSprop 均为**自适应学习率**优化器

**AdamW 四步核心**
1. 一阶动量：m_t = β₁·m_{t-1} + (1-β₁)·g_t
2. 二阶动量：v_t = β₂·v_{t-1} + (1-β₂)·g_t²
3. 偏差修正：m̂_t = m_t/(1-β₁^t)，v̂_t = v_t/(1-β₂^t)
4. 参数更新：θ_t = θ_{t-1} - α·(m̂_t/(√v̂_t + ε) + λ·θ_{t-1})

**混合精度训练（FP16）**
- 主要动机：降低内存占用 + 加速计算（利用 Tensor Core）
- FP16 最小正数 ≈ 6×10^-8，小梯度容易**下溢（Underflow）**
- **Loss Scaling**：放大 loss 避免梯度下溢（非梯度爆炸）

**Conv + BN 融合**
- **编译期（Offline）**完成的代数等价变换
- 将 BN 的 γ、β、μ、σ 合并到 Conv 的权重和偏置中
- 推理时消除 BN 的额外开销，精度不变

**多任务学习**
- 共享主干的优势：参数复用、计算复用、跨任务表征迁移

**GPU 存储层次（访问速度从快到慢）**
- 寄存器/共享内存（SRAM，片上）→ L2 缓存 → HBM（片外高带宽显存）→ 系统内存

## 🌟 5. Transformer 与注意力机制

### 核心要点

**标准 Self-Attention**
- Q, K, V 均来自同一输入
- Attention(Q,K,V) = softmax(QK^T/√d_k)·V

**Encoder-Decoder Attention（交叉注意力）**
- **Query** 来自解码器上一层的输出
- **Key, Value** 来自编码器的最终输出

**注意力机制优化**
- **滑动窗口注意力（SWA）**：
  - 每个 token 只关注窗口内的 token，KV Cache 更小（非全注意力大小）
  - 多层堆叠可扩大感受野，能捕获超窗口的长距离依赖
- **Flash Attention**：IO 优化（分块计算，减少 HBM 访问），计算复杂度仍为 O(n²)，不是低秩近似
- **线性注意力**：推理时可转化为 **RNN 递推**形式，每个 token 计算量与上下文长度无关

**KV Cache**
- 在推理阶段缓存历史 token 的 K, V 向量，避免重复计算
- 显存占用与**序列长度成正比**（PagedAttention 优化管理方式，但不改变正比关系）

## 🏗️ 6. 大模型

### 核心要点

**分布式训练**
- **数据并行（DDP）**：每个 GPU 独立计算梯度，迭代结束通过 **All-Reduce** 同步
  - Broadcast：单源广播；All-Gather：拼接；Send/Recv：点对点
- **张量并行（TP）**：同一层的权重矩阵按行/列切分到多个 GPU
  - Megatron-LM MLP 策略：第一层**列切分**，第二层**行切分**
- **流水线并行（PP）**：按网络层深度切分，不同层在不同 GPU
  - **Pipeline Bubble**：GPU 等待上游 Stage 时的空闲状态
- **ZeRO（DDP 增强）**：按优化器状态切分

**量化**
- **SmoothQuant**：将 activation 的大 scale 数学等价地迁移到权重上，使 activation 更易量化
  - 核心：把 activation scale 转移到权重（不是反过来）
- **INT8/FP16**：常见量化精度

**推理优化**
- **KV Cache Swap**：显存不足时将低优先级请求的 KV Cache 换出到 CPU
- **PagedAttention（vLLM）**：将 KV Cache 分页存储，利用离散显存碎片，减少浪费
  - 优化的是碎片化管理，KV Cache 占用仍与序列长度正相关
- **TTFT 飙升但 decode 速度不变** → 瓶颈在 prefill 阶段
  - 解决：摘要压缩链路（缩短输入）、开启 prefix cache

**RAG（检索增强生成）**
- **MIPS（最大内积搜索）**：计算 query 与文档的内积，选择最大的
- 通信领域 RAG：专用词表 + 按语义块切分 + 领域微调 Embedding

**多模态大模型**
- 视觉-语言连接器（Connector）：
  - **MLP**（LLaVA）：多层感知机
  - **线性投影层**：最简单的连接方式
  - **Q-Former**（BLIP-2）：可学习查询向量提取定长视觉特征

**RLHF（人类反馈强化学习）**
- SFT 阶段：监督学习模仿高质量示例
- RL 阶段：基于奖励/偏好信号进一步优化行为（不能互相替代）

**评测**
- **LLM-as-a-judge**：存在**长度偏见**，倾向于给更长的输出打高分
- **长上下文评测**：Needle-in-a-haystack + multi-hop 题型
- **Pass@1**：单次生成正确率，不代表鲁棒性

## 📈 7. 数值计算

### 核心要点

**牛顿迭代法**
- 公式：x_{n+1} = x_n - f(x_n)/f'(x_n)

**数值微分误差阶数**
- 前向差分：O(h)
- **中心差分**：O(h²)（对称结构消去一阶误差）
- **Richardson 外推**：O(h⁴）
- 注意：步长过小时浮点舍入误差反而增大

**蒙特卡洛积分 vs 黎曼积分**
- 蒙特卡洛：收敛速度 O(N^{-1/2})，与维度无关，适合**高维积分**
- 黎曼（一维等距）：收敛速度 O(N^{-2})，高维会有维度灾难

### 第1题：相关系数

## 1. 问题内容

相关系数 r = -1 说明 X 和 Y：
A. 相互独立  B. 存在线性函数关系  C. 完全负线性相关  D. 完全不相关

## 答案

### 核心要点

**C. 完全负线性相关**

### 详细说明

r = -1 表示两变量满足严格的负线性函数关系 `y = ax + b（a < 0）`，即完全负线性相关。

### 注意事项

r = 0 只表示线性不相关，不代表独立（可能存在非线性关系）。

### 第2题：SVD 与矩阵秩

## 1. 问题内容

对任意实矩阵 A，若 SVD 为 `UΣV^T`，则 A 的秩等于：
A. Σ的迹  B. Σ对角线上非零奇异值的个数  C. U的行数  D. V的列数

## 答案

### 核心要点

**B. Σ 对角线上非零奇异值的个数**

### 详细说明

矩阵的秩等于其非零奇异值的个数，是 SVD 的基本性质。Σ 的迹是所有奇异值之和，与秩无关。

### 第3题：混合精度 Loss Scaling

## 1. 问题内容

混合精度训练中 loss scaling 主要解决：
A. 梯度爆炸  B. 梯度下溢  C. 激活饱和  D. 权重过大

## 答案

### 核心要点

**B. 梯度下溢（Gradient Underflow）**

### 详细说明

FP16 能表示的最小正数约为 6×10^-8，反向传播中小梯度容易下溢为 0，Loss Scaling 通过放大 loss 避免该问题。

### 第4题：LN vs RMSNorm

## 1. 问题内容

关于 Layer Normalization 说法错误的是：
A. 可分 Pre-LN 和 Post-LN
B. 与 RMSNorm 相比，LN 去掉了减均值部分
C. LN 先算均值和方差再归一化
D. LN 可避免梯度消失/爆炸

### 核心要点

**B 说法错误（题目问哪项错误，所以答 B）**

### 第5题：线性注意力

## 1. 问题内容

关于注意力机制优化，正确的是：
A. SWA 无法捕获超窗口长距离依赖
B. SWA 层总显存与全注意力相同
C. Flash Attention 是低秩近似
D. 线性注意力推理时可转化为 RNN 递推

**D. 线性注意力推理时可转化为 RNN 递推**

- D 正确：线性注意力推理时每个新 token 计算量与上下文无关

### 第6题：Huber Loss

## 1. 问题内容

关于 Huber 损失函数，正确的是：
A. 平衡了 MSE 对异常值的敏感性和 MAE 在零点的不可导性
B. 存在不可导点
C. 误差小时为线性，大时为二次
D. 与 MSE 完全相同

## 答案

### 核心要点

**A. 平衡了 MSE 对异常值的敏感性和 MAE 在零点的不可导性**

### 详细说明

Huber Loss：
- 误差小时：用**平方损失**（光滑可导）
- 误差大时：用**线性损失**（对异常值不敏感）
- 处处可导，综合了两者优点

## 1. 问题内容


### 拓11：LoRA 微调

## 1. 问题内容

LoRA 的核心原理是什么？

## 答案

### 核心要点

LoRA（Low-Rank Adaptation）：通过低秩分解减少微调参数量。

### 详细说明

```
原始权重：W ∈ ℝ^(d×k)（冻结，不更新）
增量矩阵：ΔW = BA，其中 B ∈ ℝ^(d×r)，A ∈ ℝ^(r×k)，r << min(d,k)
推理时：W' = W + ΔW = W + BA
```

优点：参数量从 d×k 减少到 (d+k)×r；可快速切换不同任务的 LoRA 权重。
