## 二、线性代数

### 2.1 核心知识点速记

#### 2.1.1 矩阵基础
- **矩阵乘法**：$C = AB$，其中 $C_{ij} = \sum_k A_{ik} B_{kj}$
- **矩阵转置**：$(AB)^T = B^T A^T$
- **矩阵逆**：$AA^{-1} = A^{-1}A = I$
- **行列式**：矩阵可逆的充要条件是 $\det(A) \neq 0$

#### 2.1.2 矩阵秩
- **定义**：矩阵中线性无关的行（列）向量的最大数量
- **性质**：
  - $rank(AB) \leq \min(rank(A), rank(B))$
  - $rank(A) = rank(A^T)$
  - $rank(A) + rank(B) - n \leq rank(AB)$

#### 2.1.3 特征值与特征向量
- **定义**：$Av = \lambda v$，其中 $\lambda$ 是特征值，$v$ 是特征向量
- **特征多项式**：$\det(A - \lambda I) = 0$
- **迹**：$\sum \lambda_i = \text{tr}(A)$
- **行列式**：$\prod \lambda_i = \det(A)$

#### 2.1.4 正交矩阵
- $Q^T Q = QQ^T = I$
- $Q^{-1} = Q^T$
- 行向量和列向量两两正交且单位化

#### 2.1.5 矩阵分解
- **SVD分解**：$A = U \Sigma V^T$
- **特征分解**：$A = PDP^{-1}$（需$A$可对角化）

#### 2.1.6 二次型
- **正定**：对任意非零向量 $x$，$x^T A x > 0$
- **半正定**：$x^T A x \geq 0$
- **负定**：$x^T A x < 0$

## 1. 矩阵秩的性质

## 答案

### 核心要点

若 $A$ 为 $3 \times 4$ 矩阵，$B$ 为 $4 \times 3$ 矩阵，则 $AB$ 的秩满足 $\text{rank}(AB) \leq 3$。

### 详细说明

根据矩阵秩的性质：
- $\text{rank}(AB) \leq \min(rank(A), rank(B))$
- 由于 $A$ 是 $3 \times 4$ 矩阵，$\text{rank}(A) \leq 3$
- 由于 $B$ 是 $4 \times 3$ 矩阵，$\text{rank}(B) \leq 3$
- 因此 $\text{rank}(AB) \leq 3$

选项B（$\text{rank}(AB) = \text{rank}(BA)$）在一般情况下不成立；选项C符号错误（应为 $\leq$）；选项D公式错误。

### 选项
- A. $\text{rank}(AB) \leq 3$ ✓
- B. $\text{rank}(AB) = \text{rank}(BA)$
- C. $\text{rank}(AB) \geq \min(\text{rank}(A), \text{rank}(B))$
- D. $\text{rank}(AB) = \text{rank}(A) + \text{rank}(B) - 4$

## 2. 特征值计算

## 答案

### 核心要点

矩阵 $A = \begin{bmatrix} 2 & 1 \\ 0 & 3 \end{bmatrix}$ 的特征值为 2 和 3。

### 详细说明

特征多项式为：
$$\det(A - \lambda I) = \begin{vmatrix} 2-\lambda & 1 \\ 0 & 3-\lambda \end{vmatrix} = (2-\lambda)(3-\lambda) = 0$$

解得 $\lambda_1 = 2, \lambda_2 = 3$。

对于上三角矩阵，特征值就是对角线元素。

### 选项
- A. 1, 3
- B. 2, 3 ✓
- C. 2, 4
- D. 1, 4

## 3. 矩阵正定性判断

## 答案

### 核心要点

矩阵 $A = \begin{bmatrix} 2 & -1 \\ -1 & 2 \end{bmatrix}$ 是正定的。

### 详细说明

使用顺序主子式法判断矩阵正定性：
- $\Delta_1 = 2 > 0$
- $\Delta_2 = \det\begin{bmatrix} 2 & -1 \\ -1 & 2 \end{bmatrix} = 4 - 1 = 3 > 0$

所有顺序主子式均大于0，故矩阵正定。

### 选项
- A. 正定 ✓
- B. 负定
- C. 不定
- D. 半正定

## 6. 矩阵乘法性质

## 答案

### 核心要点

设 $A, B$ 为 $n \times n$ 矩阵，则 $(AB)^{-1} = B^{-1} A^{-1}$ 是正确的。

### 详细说明

- A错误：$(AB)^T = B^T A^T$（顺序相反）
- B正确：$(AB)^{-1} = B^{-1} A^{-1}$（当$A,B$均可逆时）
- C错误：行列式一般不满足加法 $\det(A+B) \neq \det(A) + \det(B)$
- D错误：$\text{tr}(AB) = \text{tr}(BA)$，不等于乘积 $\text{tr}(A)\text{tr}(B)$

### 选项
- A. $(AB)^T = A^T B^T$
- B. $(AB)^{-1} = B^{-1} A^{-1}$ ✓
- C. $\det(A+B) = \det(A) + \det(B)$
- D. $\text{tr}(AB) = \text{tr}(A)\text{tr}(B)$

## 三、概率论与数理统计

### 3.1 核心知识点速记

#### 3.1.1 概率基础
- **加法公式**：$P(A \cup B) = P(A) + P(B) - P(A \cap B)$
- **乘法公式**：$P(A \cap B) = P(A) P(B|A)$
- **独立**：$P(AB) = P(A)P(B)$
- **贝叶斯公式**：$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$

#### 3.1.2 随机变量分布

| 分布 | 记号 | 期望 | 方差 |
|------|------|------|------|
| 0-1分布 | Bernoulli(p) | p | p(1-p) |
| 二项分布 | B(n,p) | np | np(1-p) |
| 泊松分布 | P(λ) | λ | λ |
| 指数分布 | Exp(λ) | 1/λ | 1/λ² |
| 正态分布 | N(μ,σ²) | μ | σ² |
| 卡方分布 | χ²(n) | n | 2n |
| t分布 | t(n) | 0 | n/(n-2) |
| F分布 | F(m,n) | n/(n-2) | 复杂 |

#### 3.1.3 大数定律与中心极限定理
- **大数定律**：样本均值依概率收敛于期望
- **中心极限定理**：$n \to \infty$ 时，$\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \to N(0,1)$

#### 3.1.4 估计方法
- **矩估计**：用样本矩估计总体矩
- **最大似然估计（MLE）**：找到使似然函数最大的参数值

## 9. 贝叶斯公式

## 答案

### 核心要点

某种疾病在人群中的发病率为0.01%，检测对该疾病的准确率为99%。某人检测结果为阳性，则该人实际患病的概率约为 1%。

### 详细说明

设 $A$=患病，$B$=阳性。
$$P(A|B) = \frac{P(B|A)P(A)}{P(B)} = \frac{0.99 \times 0.0001}{0.99 \times 0.0001 + 0.01 \times 0.9999} \approx \frac{0.000099}{0.000099 + 0.009999} \approx 0.0098 \approx 1\%$$

这是典型的贝叶斯问题，注意先验概率很小（0.01%），即使检测准确率99%，后验概率仍然很低。

### 选项
- A. 99%
- B. 50%
- C. 约1% ✓
- D. 约0.01%

## 10. 正态分布标准化

## 答案

### 核心要点

设 $X \sim N(100, 16)$，则 $P(X > 108) = 0.0228$。

### 详细说明

标准化 $Z = \frac{X - \mu}{\sigma} = \frac{X - 100}{4} \sim N(0,1)$
$$P(X > 108) = P(Z > \frac{108-100}{4}) = P(Z > 2) = 1 - \Phi(2) \approx 1 - 0.9772 = 0.0228$$

标准正态分布表：$\Phi(2) \approx 0.9772$

### 选项
- A. 0.0228 ✓
- B. 0.9772
- C. 0.8413
- D. 0.1587

## 12. 最大似然估计

## 答案

### 核心要点

设 $X_1, X_2, ..., X_n$ 是来自正态分布 $N(\mu, \sigma^2)$ 的样本，其中 $\sigma^2$ 已知，则 $\mu$ 的最大似然估计为样本均值 $\bar{X}$。

### 详细说明

正态分布的对数似然函数为：
$$L(\mu) = \prod_{i=1}^n \frac{1}{\sqrt{2\pi\sigma^2}} e^{-\frac{(X_i-\mu)^2}{2\sigma^2}}$$

求导并令其为0：
$$\frac{d \ln L}{d\mu} = \sum_{i=1}^n \frac{X_i - \mu}{\sigma^2} = 0$$

解得：$\hat{\mu} = \frac{1}{n}\sum_{i=1}^n X_i = \bar{X}$

### 选项
- A. $\frac{1}{n}\sum X_i$ ✓
- B. $\max(X_1,...,X_n)$
- C. $\frac{1}{n}\sum X_i^2$
- D. $\frac{\sum (X_i - \bar{X})^2}{n}$

## 13. 协方差性质

## 答案

### 核心要点

若 $X, Y$ 相互独立，则 $\text{Cov}(X, Y) = 0$。

### 详细说明

协方差定义：$\text{Cov}(X,Y) = E(XY) - E(X)E(Y)$

独立时 $E(XY) = E(X)E(Y)$，故：
$$\text{Cov}(X,Y) = E(XY) - E(X)E(Y) = 0$$

注意：协方差为0不能推出独立（只能推出不相关），但独立一定推出协方差为0。

### 选项
- A. 1
- B. -1
- C. 0 ✓
- D. 无法确定

## 16. 假设检验

## 答案

### 核心要点

对正态总体 $N(\mu, \sigma^2)$ 进行假设检验，$H_0: \mu = \mu_0$，$H_1: \mu > \mu_0$，则在 $H_0$ 成立的条件下，检验统计量服从 $t(n-1)$。

### 详细说明

小样本（方差未知）时，用 $t$ 检验，统计量：
$$t = \frac{\bar{X} - \mu_0}{s/\sqrt{n}} \sim t(n-1)$$

若方差已知则服从 $N(0,1)$；若大样本则可用 $Z$ 近似。

### 选项
- A. $N(0,1)$
- B. $t(n-1)$ ✓
- C. $\chi^2(n-1)$
- D. $F(m,n)$

## 四、机器学习基础

### 4.1 核心知识点速记

#### 4.1.1 基本概念
- **监督学习**：有标签的学习（分类、回归）
- **无监督学习**：无标签的学习（聚类、降维）
- **强化学习**：通过奖励反馈学习策略
- **过拟合**：训练误差小，测试误差大
- **欠拟合**：训练误差和测试误差都大
- **偏差-方差分解**：泛化误差 = 偏差² + 方差 + 噪声

#### 4.1.2 常用算法

| 算法 | 类型 | 核心思想 |
|------|------|---------|
| 线性回归 | 回归 | 最小二乘法 |
| 逻辑回归 | 分类 | sigmoid函数 |
| 决策树 | 分类/回归 | 信息增益/基尼系数 |
| SVM | 分类 | 最大化间隔 |
| 朴素贝叶斯 | 分类 | 贝叶斯定理 |
| KNN | 分类/回归 | k近邻 |
| K-Means | 聚类 | 距离最小化 |
| PCA | 降维 | 方差最大化 |

#### 4.1.3 模型评估指标

**分类指标**：
- **准确率**：$\frac{TP + TN}{TP + TN + FP + FN}$
- **精确率**：$frac{TP}{TP + FP}$
- **召回率**：$\frac{TP}{TP + FN}$
- **F1分数**：$\frac{2 \times Precision \times Recall}{Precision + Recall}$

## 21. PCA降维目标

## 答案

### 核心要点

PCA降维的主要目标是**减少特征数量同时保留最大方差信息**。

### 详细说明

- 将数据投影到方差最大的方向
- 实现降维同时保留最多信息

## 五、深度学习与神经网络

### 5.1 核心知识点速记

#### 5.1.1 激活函数
- **Sigmoid**：$\sigma(x) = \frac{1}{1+e^{-x}}$
- **Tanh**：$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$
- **ReLU**：$\max(0, x)$
- **Softmax**：$softmax(x)_i = \frac{e^{x_i}}{\sum_j e^{x_j}}$

#### 5.1.2 反向传播
- **链式法则**：计算梯度
- **梯度消失**：sigmoid/tanh导数小于1，层数深时梯度衰减
- **梯度爆炸**：权重初始化过大或层数过深导致梯度过大

#### 5.1.3 CNN核心概念
- **卷积层**：局部连接、权值共享
- **池化层**：下采样（Max Pooling、Average Pooling）
- **感受野**：输出特征图上的像素对应的输入图像的区域大小

#### 5.1.4 RNN核心概念
- **LSTM**：长短期记忆门控机制（遗忘门、输入门、输出门）
- **梯度问题**：长序列梯度消失/爆炸

## 答案

### 核心要点

- **Tanh**：可用于隐藏层，但有梯度消失问题

## 26. 梯度消失原因

## 答案

### 核心要点

Sigmoid函数作为激活函数时，容易出现梯度消失的主要原因是**导数值域在(0, 0.25]之间**。

### 详细说明

Sigmoid函数：
- $\sigma(x) = \frac{1}{1+e^{-x}}$
- 导数 $\sigma'(x) = \sigma(x)(1-\sigma(x))$，最大值为0.25
- 多层反向传播时，多个小于1的数相乘，梯度迅速衰减趋近于0

这就是为什么深层网络难以训练的原因之一。

### 选项
- A. 函数值域范围太小
- B. 导数值域在(0, 0.25]之间，层数深时累积相乘趋近于0 ✓

- D. 函数关于原点不对称

## 50. 探索-利用困境

## 答案

### 核心要点

强化学习中的"探索-利用困境"是指**在尝试新动作和利用已知最优动作之间权衡**。

### 详细说明

探索-利用困境（Exploration-Exploitation Dilemma）：
- **探索**：尝试新的未知动作，可能获得更好的长期回报
- **利用**：使用已学习到的最优动作，最大化即时回报
- 智能体需要在两者之间做出权衡
- 常用方法：ε-greedy、UCB、Thompson采样

### 选项
- A. 探索环境与利用资源之间的矛盾
- B. 在尝试新动作（探索）和使用已知最优动作（利用）之间权衡 ✓
- C. 在训练和测试之间权衡
- D. 在精度和速度之间权衡
