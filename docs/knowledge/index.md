# 知识库总入口

本目录包含项目的知识文档，作为 AI Agent 工作的参考。

## 目录结构

```
knowledge/
├── index.md              # 本文件 - 知识库总入口
├── architecture.md      # 技术栈文档
├── project-context.md   # 项目上下文
├── domain-mapping.md    # 领域映射表
├── service-meta.yaml # 服务元数据 (仅后端服务)
├── service-boundary.md # 服务边界 (仅后端服务)
└── prd/             # 需求文档
    └── index.md     # 需求索引
└── exp/            # 经验沉淀
    └── index.md   # 经验索引
```

## 知识库索引

### 技术栈

- `architecture.md` - 技术栈、框架、中间件

### 项目信息

- `project-context.md` - 项目上下文

### 需求与经验

- `prd/index.md` - 需求文档索引
- `exp/index.md` - 经验文档索引

## 使用说明

1. 先读 `index.md` 建立知识地图
2. 再进入相关索引或稳定入口
3. 最后按需读取叶子文档
